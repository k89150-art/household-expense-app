# 一起記

iPhone 優先的家庭收支 App，使用 Next.js + Firebase + Vercel。

## 核心特色

- 夫妻共用家庭帳本
- 不做分帳，專注家庭收支
- 月初生活費轉帳紀錄
- 支出、收入、固定支出模板
- 醫療費可分對象
- 管理費、房貸、保險、水電瓦斯、網路費
- 信用卡消費逐筆記，帳單只追蹤繳款
- 個人雜支私人明細
- 醫院午餐預付金每週結算

## 開發

```bash
npm install
npm run dev
```

複製 `.env.example` 成 `.env.local`，填入 Firebase 專案設定。
