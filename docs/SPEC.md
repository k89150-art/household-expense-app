# 夫妻家庭帳本正式版規格

## 產品定位

iPhone 優先的夫妻共用家庭記帳 App。核心不是 AA 分帳，而是家庭收支、生活費轉帳、固定支出、信用卡、預付金與個人隱私支出的管理。

## 技術架構

- 前端：Next.js / React / TypeScript
- 資料庫：Firebase Firestore
- 登入：Firebase Authentication
- 部署：Vercel
- 使用方式：iPhone Safari 開啟後加入主畫面

## 主要功能

1. 登入與家庭帳本
2. 你與太太兩位成員共同使用
3. 新增、修改、刪除支出
4. 新增、修改、刪除收入
5. 月初生活費轉帳紀錄
6. 固定支出模板：保險、房貸、管理費、網路費、訂閱
7. 固定支出可每月點選帶入
8. 固定支出支援「只改本月」與「修改模板，之後月份套用」
9. 醫療費可選消費歸屬：Chris、太太、兒子、貓、家庭
10. 信用卡消費一筆一筆記，信用卡帳單只做繳款追蹤，不列入支出統計
11. 個人雜支：共同帳本只顯示個人雜支，私人明細只有本人可見
12. 醫院午餐預付金：每週輸入剩餘金額，自動反推本週餐費

## 醫院午餐預付金邏輯

書記每週告知剩餘金額，不需每天記午餐。

公式：

```text
本週午餐支出 = 上次餘額 + 本週加值 - 本次餘額
```

例如：

```text
上次餘額：1,000
本週加值：0
本次剩餘：580
本週午餐支出：420
```

系統自動建立：

```text
類別：餐飲
歸屬：Chris
付款方式：醫院午餐預付金
金額：420
備註：本週醫院午餐
```

## 個人雜支邏輯

不做假帳、不平均灌到其他分類。

共同帳本可見：

```text
日期、金額、類別：個人雜支、支付者
```

本人登入後可見：

```text
實際品項、私人備註
```

私人明細拆到 privateExpenseDetails collection，透過 Firebase Security Rules 限制只有 owner 可讀。

## 信用卡邏輯

- 刷卡消費：一筆一筆記，付款方式選信用卡
- 訂閱/保險/網路費：做成固定支出模板，每月確認帶入
- 信用卡繳款：記錄已繳/未繳，但不列入支出統計，避免重複計算
